# SlowedReverb — онлайн‑сервис для создания slowed + reverb треков

## Общая концепция

SlowedReverb — это веб‑приложение, где любой пользователь может загрузить аудио и за пару кликов получить версию с эффектами **slowed** и **reverb**. Интерфейс максимально простой: загрузил трек, отрегулировал скорость и уровень реверба ползунками, послушал предпросмотр и при желании скачал готовый файл.

Главная идея проекта — сделать удобный онлайн‑инструмент без лишних сложностей, доступный как на компьютере, так и на мобильных устройствах.

---

## Страницы сайта

1. **Главная (Home)**
   Краткое описание сервиса и ссылки на основные вкладки.

2. **Редактор (Upload + Editor)**
   Загрузка трека и редактирование на одной странице. Здесь есть плеер, ползунок для скорости, ползунок для реверба и кнопки предпросмотра/экспорта.

3. **Библиотека (Library)**
   Хранение обработанных треков пользователя. Можно вернуться, переслушать или скачать.

4. **Аккаунт (Account)**
   Регистрация, вход и управление профилем.

---

## Сущности базы данных

Чтобы хранить только необходимое, выделены три сущности:

* **users**
  id (PK), name, email, password\_hash, role (user/admin), created\_at

* **tracks** — загруженные исходные файлы
  id (PK), user\_id (FK → users.id), original\_filename, storage\_key, duration\_seconds, created\_at

* **processed\_tracks** — готовые версии после обработки
  id (PK), track\_id (FK → tracks.id), user\_id (FK → users.id), speed\_factor, reverb\_amount, storage\_key, created\_at

---

## Особенности проекта

* Минимум вкладок и сущностей, чтобы не перегружать интерфейс.
* Поддержка только двух базовых эффектов: **slowed** и **reverb**.
* Предпросмотр работает прямо в браузере через WebAudio API.
* Для скачивания итогового файла используется серверная обработка на базе FFmpeg.
